EmasLogic

trait EmasLogic extends DelegatingLogicStrategy with EmasPopulation with EmasBehaviour with EmasMeetings with DefaultFight with DefaultReproduction with EmasStats

Default EMAS logic. Combines the default strategies for generating the initial population, agent behaviour and meetings, as well as default EMAS statistics.
